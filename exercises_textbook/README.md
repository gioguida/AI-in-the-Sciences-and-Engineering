# Exercises Textbook

This folder contains LaTeX source files for a textbook of exercises related to AI in the Sciences and Engineering.

## Contents

- `exercises_textbook.tex` - Main LaTeX document
- `chapters/` - Individual chapter files
  - `chapter01_tensors.tex` - Chapter on tensors

## Status

⚠️ **Work in Progress** - This project is currently in development and will likely remain incomplete.

## Building

To compile the document:

```bash
pdflatex exercises_textbook.tex
```
